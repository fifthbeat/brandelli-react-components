## Description

A radio switcher that display content

## How to use

Component example:

```js
const displayContent = [{id: 'item1', label: 'Item 1'}, {id: 'item2', label: 'Item 2'}, {id: 'item3', label: 'Item 3'}];

<Switcher content={displayContent} header="Header text" footer="Footer text" />;
```
