## How to use

React component example:

```js
<List border={true} direction={'column'} size={'large'} maxHeight={'50px'}>
  <li>test 1</li>
  <li>test 2</li>
  <li>test 3</li>
  <li>test 4</li>
  <li>test 5</li>
  <li>test 6</li>
</List>
```

List with items

```js
const Item = require('./Item');
<List border={true} direction={'column'} size={'large'} maxHeight={'50px'}>
  <Item />
</List>;
```
